UI AND UX STANDARDS

Hospital care startup

In this project for a Startup for hospital care, the challenge was to develop an experience for hospital care, combining agility and humanization

RESPONSIBILITIES

UX Design

UI Design

HTML & CSS code

Focus on user needs

The first step was to conduct user research to identify the pain points and needs of the target audience. I used this information to create user personas and user journey maps that accurately reflect the user experience.

Use a human-centered design approach

When I was designing the user experience, I tried to focus on empathy and understanding the user's perspective. I incorporate the feedback and iterate on designs based on user testing.

Also, the other challenge was creating a service blueprint: In this case, the service blueprint helped to visualize the user experience and identify areas for improvement. One of the most interesting things about this case was the voice commander filling the fields on forms.

The most important thing, in this case, was the prioritize agility and humanization: We try to find a way to balance agility and humanization in hospital care. So, we try to design a system that is both efficient and compassionate. Consider using technology to streamline processes while still prioritizing the human connection between patients and care providers.
