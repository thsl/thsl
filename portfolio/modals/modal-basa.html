UX STUDIES APPLIED IN THIS PROJECT

Project developed with agile methods

UX Discovery Research:
This project kicked off with a comprehensive UX discovery phase. By conducting thorough research, I delved into the needs, preferences, and pain points of the app's diverse user base. This step laid the foundation for informed design decisions and a user-centric approach.

Interaction Design & Information Architecture:
A core aspect of the project involved crafting seamless interactions that seamlessly guided users through their banking tasks. With a focus on intuitive navigation and task flow, I optimized the app's information architecture to ensure effortless access to key functionalities.

User Interface Design:
Translating insights from the discovery research into visually appealing interfaces was a pivotal aspect of the project. My role encompassed designing interfaces that aligned with the brand's visual identity while prioritizing clarity and ease of use.

Usability Testing with Prototypes:
To validate design concepts, I developed interactive prototypes that simulated the app's interactions. These prototypes were subjected to rigorous usability testing, allowing me to gather valuable user feedback and make iterative improvements to the designs.

Collaboration with Developers:
A successful UX project relies on seamless collaboration with developers. I fostered open lines of communication to ensure that the design vision was translated into a functional digital product. This collaborative approach minimized friction and streamlined the development process.

HTML and CSS Customization:
Going the extra mile to ensure a cohesive user experience, I provided support in HTML and CSS customization. This hands-on approach allowed me to fine-tune design elements and maintain design integrity during the implementation phase.
